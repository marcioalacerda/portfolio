Habilidades

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web.

HTML é uma linguagem de marcação utilizada na construção de páginas na Web.

CSS é um mecanismo para adicionar estilo a um documento web.

SASS é uma linguagem de extensão do CSS, a sua ideia é adicionar recursos especiais como variáveis, mixins, funções e operações e outras opções

Less é uma linguagem de folha de estilos dinâmica desenhada por Alexis Sellier. Ela foi influenciada por SASS e influenciou a nova sintaxe "SCSS" do SASS, que adaptou sua sintaxe de formação de blocos do tipo CSS.

Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.

GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git.

React é uma biblioteca criada para construir telas de forma declarativa. Em desenvolvimento.

AngularJS é um framework open-source de desenvolvimento front-end que possibilita o desenvolvimento de aplicações web. Em desenvolvimento.

TypeScript é um superconjunto de JavaScript, ou seja, um conjunto de ferramentas e formas mais eficientes de escrever código JavaScript. Em desenvolvimento.
